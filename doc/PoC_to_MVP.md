# MVP-Ziel

## MVP-Ziel (einfach erklärt)
Unser MVP ist erreicht, wenn Nutzer:
- eine Karte mit Zecken-Hotspots sehen können
- Zecken melden können
- aktuelle Daten in Echtzeit erhalten
- die App grundlegend bedienen können
- und die Daten DSGVO-konform gespeichert sind

Alles darüber hinaus ist nicht zwingend MVP, sondern Verbesserung.

---

## Welche Features sind für den MVP notwendig?

### MVP-Pflicht-Features (Backlog Items)
Diese Features müssen umgesetzt werden, damit wir den MVP veröffentlichen können.

#### Technische Basis
- Datenbank einrichten (#19)
- DSGVO-konforme Datenhaltung (#15)

#### App-Grundlagen
- Main Page Grundstruktur (#20)
- Navigation (#7)

#### Kernfunktionen
- Karte mit Zecken-Hotspots (#12)
- Zeckenmeldung erfassen (#13)
- Meldungen in Echtzeit anzeigen (#10)

#### Basis-Mehrwert
- Erste-Hilfe-Info-Seite (#9)

Ohne diese Punkte kein MVP.
